# Boolean Algebra Solver

A small JavaScript project that parses and evaluates Boolean expressions using Chevrotain

This project supports logical operations including:

- `and`  
- `or`  
- `not`  
- Implication `=>`  
- Equivalence `<=>`  
- Parentheses

---

## Usage

To run the React application, run the following command
```bash
npm start 
```
